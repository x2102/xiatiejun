Welcome to Tiejun Blog

9月 11, 2017

这几天突然的一个想法, 就是建立自己的博客. 以前工作中遇到的一些问题, 虽然都解决了, 但是现在在记忆中已经很淡了. 以后遇到了,知道解决过, 但是过程还是要去找, 感觉像是浪费了时间. 现在有了自己的博客, 可以把工作的遇到的一些问题, 包括学习到的一些新的知识记录下来.可以做一个很好的总结,加深印象, 希望以后可以帮助从事iOS行业的开发者.自己都是从一个一个坑里走出来的, 能体会初学的那种感受.
最后再想想用自己的名字做域名的,想想都觉得开心,当然也是为了zb.

建立博客的简单过程

我是在mac环境下 使用的是hexo + GitHub page 搭建的, 大概的步骤:
1, 安装node.js
2, 安装git(xcode自带git, 可以省略)
3, 初始化hexo
4, 注册github 账号
5, 建立自己的代码仓库
6 部署到GitHub
7, 购买属于自己的域名
8, 在购买域名的网站上添加解析(note: 如果是在阿里的万网买的话, 购买成功后会有很多的解析过的域名的列表,这些没有什么用, 如果不删除的话, 后面添加GitHub 的上的仓库地址的话, 就会添加不了)
9, 在搭建好的hexo的环境下的source的_post文件添加一个CNAME文件, 上面写上购买的域名,不然就关联不了自己的GitHub的网页
