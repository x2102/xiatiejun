Cornerstone提示已损坏打不开

Sep 12, 2017

问题原因

软件有经过了汉化或者破解，所以可能被Mac认为「已损坏」

解决办法

系统偏好设置 -> 安全性与隐私 -> 通用 -> 选择“任何来源”, 如果没有看见任何来源, 在控制台执行 sudo spctl --master-disable, 不显示任何来源执行 sudo spctl --master-enable
